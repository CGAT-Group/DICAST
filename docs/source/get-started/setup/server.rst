ssh connection GUI support
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

If you connect to your server via ssh and want to use the sequanix GUI to run the pipeline
make sure to use the -X option to enable forwarding of

.. prompt:: bash $

	ssh -X *destination*

.. note::

	If you want to use DICAST on a server via ssh from a Windows system and want to
	use the sequanix GUI, make also sure you have an X11 server installed and running.
